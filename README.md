# Taokim Homebrew Tap

This is a [Homebrew](https://brew.sh) tap for software by [@taokim](https://github.com/taokim).

## Installation

```bash
brew tap taokim/tap
```

## Available Formulae

### repo-claude

Multi-agent orchestration using Repo tool and Claude Code.

```bash
brew install taokim/tap/repo-claude
```

Or install directly:

```bash
brew install taokim/tap/repo-claude
```
